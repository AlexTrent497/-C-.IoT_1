#Правила MD

Для заголовка перед словам с лева ставим решётку

Для курсива заключаем слово в звёздочки

Для полужирного заключаем слово в двойные звёздочки

Для простого списка ставим звёздочку пробел с лева перед элементом списка

Более мелкии заголовки выделяются двумя решётками слева

Для пронумерованого списка ставят номер точку и пробел слева от его элемента
